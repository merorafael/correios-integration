Integration with [Correios](http://www.correios.com.br/) Webservices.

Requirements
------------

- PHP 5.4 or above
- SOAP extension

Instalation with composer
-------------------------

1. Open your project directory;
2. Run `composer require mero/correios` to add `Mero Correios`
 in your project vendor.
